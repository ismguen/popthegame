# Dew Drop Pop Agent Pack

Dieses ZIP enthält für **jede wichtige reale Studio-Rolle** einen eigenen Markdown-Agenten, angepasst auf dein Spiel.

## Inhalt
- `/agents/*.md` — einzelne Agenten
- `AGENT_INDEX.md` — Übersicht
- `MASTER_ORCHESTRATOR.md` — zentrale Koordination
- `TEAM_MATRIX.md` — wer mit wem arbeitet
- `PROMPT_STARTER.md` — schnelle Startprompts
- `PROJECT_CONTEXT.md` — zusammengefasster Projektkontext

## Nutzung
1. Öffne `MASTER_ORCHESTRATOR.md`
2. Wähle die passenden Agenten
3. Gib einem Agenten ein konkretes Problem
4. Lass benachbarte Agenten reviewen
5. Übersetze Ergebnisse in Produktionsaufgaben

## Empfohlener Start
Für dein Spiel zuerst diese Agenten verwenden:
- Game Director
- Core Gameplay Designer
- Level Designer
- Economy Designer
- UI Designer
- Gameplay Programmer
- QA Lead
- Data Analyst
